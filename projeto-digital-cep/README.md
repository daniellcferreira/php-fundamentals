## Descrição
Projeto para consulta de CEP utilizando a API ViaCEP.  
Permite buscar informações de endereço a partir de um CEP informado, retornando os dados em formato de array.  
Inclui testes automatizados com PHPUnit.

## Funcionalidades
- Consulta de CEP pela API ViaCEP.
- Sanitização automática do CEP.
- Retorno estruturado em array.
- Testes automatizados com PHPUnit.
- Suporte a autoload via Composer.
