## Descrição

Este projeto foi desenvolvido com fins **educativos** para demonstrar o uso da **extensão PDO (PHP Data Objects)** na interação com bancos de dados MySQL.  
O objetivo é mostrar de forma prática como realizar operações básicas de **CRUD** — *Create, Read, Update e Delete* — utilizando código simples, organizado e seguro.

---

## Funcionalidades

- **Conexão com o banco de dados MySQL** via PDO.  
- **Listagem de produtos** armazenados no banco (`list.php`).  
- **Inserção de novos produtos** (`insert.php`).  
- **Atualização de produtos existentes** (`update.php`).  
- **Exclusão de produtos** (`delete.php`).  
- Tratamento básico de exceções e uso de `prepare` e `bindParam` para segurança.
