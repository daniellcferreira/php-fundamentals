## Descrição

Este projeto demonstra a aplicação de **Programação Orientada a Objetos (POO)** em PHP através de um **sistema bancário simples**.  
Foram implementadas classes para representar diferentes tipos de contas — **Conta Corrente** e **Conta Poupança** — com comportamentos distintos e uso de **herança**, **encapsulamento**, **abstração** e **métodos polimórficos**.

## Funcionalidades

- Criação de contas bancárias com dados personalizados.  
- Depósito e saque de valores.  
- Exibição do saldo com formatação monetária.  
- Regras específicas por tipo de conta:
  - **Conta Corrente:** aplica taxa fixa de operação.  
  - **Conta Poupança:** aplica rendimento sobre o saldo.  
- Scripts de depuração para testar o comportamento de cada tipo de conta.
